# 🚀 LifeHub

## 📌 Proje Hakkında

**LifeHub**, kullanıcıların aboneliklerini, faturalarını ve bütçelerini tek bir platform üzerinden yönetmelerini sağlayan, yapay zekâ destekli kişisel yaşam yönetim uygulamasıdır.

Projenin amacı; unutulan ödemeleri azaltmak, finansal farkındalık oluşturmak ve kullanıcıya kişiselleştirilmiş öneriler sunmaktır.

---

## 🎯 Hedefler

* Abonelik yönetimi
* Fatura takibi
* Bütçe yönetimi
* Hatırlatma sistemi
* Yapay zekâ destekli haftalık özetler
* Kullanıcı dostu dashboard

---

## 🛠️ Kullanılacak Teknolojiler

### Backend

* Java
* Spring Boot
* Spring Data JPA
* PostgreSQL

### Frontend

* React
* Vite

### Diğer

* Docker
* Redis
* RabbitMQ
* JWT
* OpenAI API
